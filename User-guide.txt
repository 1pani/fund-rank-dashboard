1.Install latest python 3 version for windows - https://www.python.org/downloads/
2."pip install <package-name>" or "python -m pip install <package-name>"
3.Clone the following repository and download it from - https://github.com/1pani/fund-rank-dashboard
4.Install the following - pip install dash==0.21.1  # The core dash backend
						python -m pip install dash-renderer==0.13.0  # The dash front-end
						python -m pip install dash-html-components==0.11.0  # HTML components
						python -m pip install dash-core-components==0.23.0  # Supercharged components
						python -m pip install plotly --upgrade  # Plotly graphing library used in examples
						python -m pip install dash_table_experiments
						python -m pip install pandas
						python -m pip install numpy
						python -m pip install json
5.Run the "app.py" downloaded from the repository in step 6.
6.Input the values, and follow the local ip generated at the end of the execution process.
7.You get your mutual-fund-ranking dashboard.
8.Also visit https://github.com/pettarin/python-on-windows for further help in installing python.
9.Copy the path of your excel file downloaded with the project to the input step in "app.py".
